== Description ==

This plugin will simply allow you to replace the "Howdy" text in the top right corner of your dashboard with "Welcome". All you have to do is upload the plugin and Howdy will change to Welcome in the top right of your dashboard.

If you would like to customize a little further, learn how to customize [replace howdy with your own words](http://www.cohesivewebdesigns.com/wordpress-howdy-how-to-change-wordpress-howdy-to-welcome-in-the-admin-bar/). 

This plugin was created by [Cohesive Web Design](http://www.cohesivewebdesigns.com). 

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
